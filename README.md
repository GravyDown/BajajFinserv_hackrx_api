# HackRx Document Q&A API

An intelligent document-based question-answering system optimized for legal, insurance, and compliance use cases.

## Features

- 📄 **Multi-format Support**: PDF, DOCX, Email (.eml) files
- 🔗 **URL Support**: Direct URLs and Google Drive links
- 🧠 **Smart Chunking**: Optimized text splitting for context preservation
- 🔍 **Semantic Search**: FAISS-powered vector similarity search
- 🤖 **Multi-model LLM**: Google Gemini, Groq, and HuggingFace fallback
- 📊 **Confidence Scoring**: Answer reliability metrics
- 🚀 **Fast & Scalable**: Async processing with parallel question handling
